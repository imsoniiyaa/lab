## Overview

This repository contains my implementation for **Lab 3: Tucker Tensors**, focusing on  
low‑rank tensor representations, HOSVD, Tucker truncation, and tensor arithmetic.

The goals of this lab are to:

- Compute **mode‑n unfoldings** of 3‑way tensors.
- Apply **mode‑n products** and verify commutativity across modes.
- Implement the **Higher‑Order SVD (HOSVD)** and construct Tucker decompositions.
- Perform **Tucker truncation** and analyze approximation errors.
- Represent 3D functions on fine grids in **compressed Tucker format**.
- Implement **Tucker arithmetic** (addition, inner product, norm) without expanding tensors.
- Extend the **Step‑and‑Truncate PDE solver** to 3D advection in Tucker format.
